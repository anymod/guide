## AnyMod Guide
Docs built with [Vuepress](https://vuepress.vuejs.org/), with app located at:

| ENV | URL |
|---|---|
| `production`  | https://guide.anymod.com |
| `development`  | http://localhost:8080 |

## Operations

### Run local server
`npm run docs:dev`

### Build for production
`npm run docs:build`
